# Robotics Lab Assignments

This repository contains the completed tasks for the Robotics Lab assignments by **Rahul Rajak**.

## Repository Structure

*   **[Task1](./Task1)**: Object Detection and Pick-and-Place Operation in Simulation
    *   Demonstrates accurate object detection, back-projection of coordinates, and robust pick-and-place capabilities.
*   **[Task2](./Task2)**: Robotic Arm Motion Planning in Cluttered Environments
    *   Implements a custom **Hybrid APF-RRT Planner** for a 6-DOF robotic arm navigating through a "forest" of obstacles using ROS 2, MoveIt 2, and Ignition Gazebo.

## Getting Started

Please refer to the `README.md` files inside each task's folder for specific instructions on how to build and run them.
